## 🗂️ 归档策略

### 何时归档

文档在以下情况下会被归档:
1. 被更完整的文档替代
2. 记录的是过时的中间状态
3. 仅具有历史参考价值

### 归档原则

- ✅ 保留原始内容,不做修改
- ✅ 添加归档说明和时间戳
- ✅ 在新文档中提供替代链接
- ✅ 定期清理过于陈旧的归档
